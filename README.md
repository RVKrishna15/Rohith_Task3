# Password Generator

## Description
Password Generator is a beginner-friendly Python application that creates secure random passwords based on user preferences. Users can choose the password length and include letters, numbers, and special characters.

## Features
- Custom password length
- Letters
- Numbers
- Symbols
- Random password generation

## Technologies Used
- Python 3

## How to Run

```bash
py security.py
```
